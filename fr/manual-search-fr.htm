Recherche en ligne

The aptosid online search uses the powerful xapian-omega indexer and it is only applicable when using the online aptosid-manual.

Option 1

Saisir un ou plusieurs mots dans le champ recherche et cliquer sur Search (ou la touche Entrée).



Matching any words Matching all words

Language:

Hits per page:

Options de recherche

Lorsque Rechercher TOUS les mots est selectionné, seuls les documents contenant les termes de recherche s'afficheront. Avec Rechercher PARMI les mots tous les documents qui contiennent au moins un des termes que vous recherchez s'afficheront.

Sinon, vous pouvez mettre un signe plus (+) directement en face d'un ou plusieurs mots pour obtenir uniquement les fichiers qui incluent tous les mots. Les mots avec un signe moins (-) directement en face pour modifier le résultat de sorte que seuls les documents qui ne contiennent aucun de ces mots soient retournés.

Les résultats sont classés par pertinence avec les documents les plus pertinents en premier. La pertinence dépend du nombre et de la position des mots dans les documents correspondants.

Option 2

Pour effectuer une recherche en ligne, entrez les critères suivants dans le champ de recherche de votre navigateur (où xxxx est le mot que vous voulez rechercher) :

xxxxx site:manual.aptosid.com/fr

Recherche en local

L'équipe de manuel de aptosid a créé une configuration spéciale avec le développeur de Recoll, qui est incluse dans le paquet par défaut. Un grand merci à Recoll.

Recoll est intitulé Local Text Search au menu Utilitaires de KDE ou au menu Debian -> Applications -> Gestion de données -> Data Search Tool (Outil de recherche de données).

Étape 1

Le manuel est actuellement disponible via apt pour langues suivantes :
Default - Deutsch (German, -de)
Default - English (English, -en)
*Čeština (Czech, -cz)
*Dansk (Danish, -da)
*Español (Spanish, -es)
*Français (French, -fr)
*Italiano (Italian, -it)
*Nihongo (Japanese, -ja)
*Nederlands (Dutch, -nl)
*Polski (Polish, -pl)
*Português (Portuguese, -pt-br)
*Română (Romanian, -ro)
*Русский (Russian, -ru)
*Українська (Ukrainian, -uk)

apt-get update
apt-get install aptosid-manual-xx (où -xx correspond au code de la langue désirée. For example: aptosid-manual-en, aptosid-manual-de, aptosid-manual-es, ....)
Étape 2
apt-get install recoll
Étape 3
  • Au premier démarrage, Recoll va vous demander si vous voulez construire une base de données, cliquez sur Annuler. Une fenêtre de configuration d'indexation va alors apparaître.
  • Dans les paramètres globaux , cliquez sur le signe + (plus) puis allez dans /usr/share/aptosid-manual puis cliquez sur votre langue puis sur OK. Cliquez ensuite sur le signe - (moins) et supprimez tous les autres répertoires, y compris ~ (tilde).
  • Finalement, allez dans Fichier > Mettre l'index à jour puis cherchez le sujet que vous désirez dans le manuel aptosid.

Si vous avez besoin de reconfigurer Recoll suite à la mise à jour du manuel aptosid à l'aide d'apt, (dans une console utilisateur $):

recollindex -z

Puis dans Recoll: Fichier > Mettre à jour l'index.

Le lien 'Prévisualiser' va mettre en évidence toutes les instances du mot recherché dans le fichier. Le lien 'Ouvrir' quant à lui ouvrira Konqueror et utilisera la commande Trouver sur cette page pour une micro recherche du mot que vous avez recherché dans Recoll.
