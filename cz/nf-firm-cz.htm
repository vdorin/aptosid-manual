Seznam zdrojů a ovladačů firmware: non-free (uzavřené)

Seznam zdrojů

Jelikož obsahuje iso aptosidu pouze svobodný software podle dfsg, budete si pravděpodobně muset přidat contrib non-free do vašeho souboru /etc/apt/sources.list.d/debian.list :

mcedit /etc/apt/sources.list.d/debian.list

Jednotlivé řádky by měly vypadat nějak takto, v závislosti na výběru vašho zrcadla (podívejte se na Seznam aktuálního stavu serveru Debianu a zrcadel):

# Debian
deb http://ftp.de.debian.org/debian unstable main contrib non-free
# deb-src http://ftp.de.debian.org/debian unstable main contrib non-free

Po změně vašich zdrojů budete muset zadat apt-get update před vyhledáváním nebo instalováním nového balíčku (apt-get update je dobrý zvyk před každou insalací balíčku).

Detekce firmwaru - non-free (uzavřený)

Pro nainstalování uzavřeného firmwaru, (předpokládejem, že ste přidali contrib non-free mezi vaše zdroje):

apt-get update
apt-get install firmware-linux firmware-linux-free firmware-linux-nonfree

Potřebný firmware můžete zjistit z následujícího výčtu zařízení/firmware nebo pomocí skriptu fw-detect (zabalíčkovaného v aptosid-scripts).

$ fw-detect

Výstup z fw-detect popisuje příkazy nutné k instalaci a aktivaci firmware

Příklad:

#Detected driver that requires firmware to operate
#Follow these instructions to obtain the correct firmware
# and activate the zd1211rw driver:
apt-get update
apt-get install zd1211-firmware
modprobe -r zd1211rw
modprobe zd1211rw

Pro instalaci firmware z git repozitářů potřebujete:

apt-get install git-core

nesvobodný firmware deb na klíči

Jestli potřebujete stáhnout .deb balíčky, abyste je zkopírovali na usb-klíč a pak na jiný počítač, můžete je stáhnout buď jako zip, nebo jako tar.gz z http://cdimage.debian.org/cdimage/unofficial/non-free/firmware/sid/current/ a rozbalit je do adresáře firmware.

Měli byste vidět několik .deb souborů. Poté stáhněte firmware-linux-free*.deb a přidejte je ke všem ostatním .deb souborům do adresáře, kam jste nesvobodný firmware rozbalili. Poté je přemístěte s usb-klíčem na počítač, který nesvobodný firmware potřebuje:

dpkg -i firmware-linux-nonfree*.deb

Snažíme se dodat balíčky pro všechen legálně šiřitelný firmware z našich nesvobodných repozitářů, ale ne všichni výrobci to dovolují.
