Inhoud van de Live ISO

aptosid levert alleen Debian dfsg-free software op de live-ISO, bekijk aub deze link voor meer informatie over niet-gratis bronnen voor firmware.

De ISO is compleet gebaseerd op Debian Sid (recentelijk van de datum 2012-12-01) verrijkt en gestabiliseerd met aptosid's eigen pakketten en scripts. Het komt met de laatste verbeterde aptosid kernel, dewelke gebaseerd is op de meest recente vanilla kernel 3.6.9-rc1. ACPI en DMA zijn standaard ingeschakeld.

Een compleet manifest van elke release variant is te vinden op elke download mirror: aptosid Mirrors, Downloaden en branden.

Release ISO varianten

aptosid biedt 5 up-to-date live-ISO release varianten van Debian Sid aan en het installeren duurt meestal niet langer dan 10-20 minuten.

De varianten zijn:
  1. KDE lite 32 bit , live-ISO ongeveer 600MB:
    bevat het minimale aan applicatie's voor de KDE4 desktop. Meer applicatie's toevoegen voor je langdurig gebruik is makkelijk te doen via apt.
  2. KDE lite 64 bit , live-ISO ongeveer 600MB:
    bevat het minimale aan applicatie's voor de KDE4 desktop. Meer applicatie's toevoegen voor je langdurig gebruik is makkelijk te doen via apt.
  3. XFCE 32 bit, live-ISO ongeveer 500MB:
    bevat een volledig Desktop Omgeving (niet minimaal), en heeft alle applicatie's om direct productief te zijn. Het is kleiner dan KDE4.
  4. XFCE 64 bit , live-ISO ongeveer 500MB:
    bevat een volledig Desktop Omgeving (niet minimaal), en heeft alle applicatie's om direct productief te zijn. Het is kleiner dan KDE4.
  5. KDE-full, een combinatie van 32 bit + amd64 bit, live-ISO ongeveer 2.5GB:
    bevat een volledige suite van KDE4 applicatie's met ondersteuning voor 16 talen en inclusief de volgende dingen, kantoorsoftware toepassingen zoals LibreOffice en web browsers zoals iceweasel (Firefox)

Applicatie's en tools

Systeemeisen voor KDE4 en XFCE

AMD64

  • Processor eisen:
    • AMD64
    • Intel Core2
    • Intel Atom Series (2xx/ 3xx, N4xx/ D4xx/ D5xx; de N2xx serie ondersteunt EM64T niet)
    • VIA Nano
    • elke x86-64/ EM64T capable CPU of nieuwer
    • nieuwere 64 bit AMD Sempron en Intel Pentium 4 CPUs (let op de "lm" flag in /proc/cpuinfo of gebruik infobash -v3)
  • Geheugen eisen:
    • KDE: ≥512 MB RAM (≥1 GB RAM aanbevolen), ≥1 GB RAM voor liveapt.
    • XFCE: ≥512 MB RAM.
  • VGA grafische kaart welke tenminste een resolutie van 640x480 pixels kan weergeven.
  • Optische diskdrive of USB media.
  • ≥3 GB vrije HDD ruimte , ≥10 GB aanbevolen.
  • ondersteunde opstart (boot) methodes:
    • PC-BIOS+MSDOS partitie tabel (maximum HDD grootte <2 TB)
    • PC-BIOS+GPT partitie tabel (maximum HDD grootte <8 ZB, /boot/ moet fysisch onder 2 TB zijn in zijn geheel)
    • UEFI+GPT (maximum HDD grootte <8 ZB)

i686

  • Processor eisen:
    • Intel Pentium pro/ Pentium II
    • AMD K7 Athlon (niet K5 / K6)
    • Intel Atom Series (Z5xx/ Z6xx, N2xx)
    • VIA C3-2 (Nehemiah, niet C3 Samuel of Ezra) / C7
    • elke x86-64/ EM64T capabele CPU of nieuwer
    • de volledige i686 command set is vereist.
  • Geheugen eisen:
    • KDE: ≥512 MB RAM (≥1 GB RAM aanbevolen), ≥1 GB RAM voor liveapt.
    • XFCE: ≥512 MB RAM.
    • Met i686 is enkel 3.1 tot 3.4 GB RAM beschikbaar, amd64 (x86-64/ EM64T) wordt aangeraden voor 64 bit hardware.
  • VGA grafische kaart welke tenminste een resolutie van 640x480 pixels kan weergeven.
  • Optische diskdrive of USB media.
  • ≥3 GB vrije HDD ruimte, ≥10 GB aanbevolen.
  • ondersteunde opstart (boot) methodes:
    • PC-BIOS+MSDOS partitie tabel (maximum HDD grootte <2 TB)
    • PC-BIOS+GPT partitie tabel (maximum HDD grootte <8 ZB, /boot/ moet fysisch onder 2 TB zijn in zijn geheel)

Applicatie's en Tools

Op de aptosid-ISO zit een variteit aan verschillende programma's en tools waarmee je bijna alles voor het dagelijks gebruik nodig hebt.

Keuze uit 2 vooraanstaande Desktop Managers voor toegang tot de programma's/toepassingen:
KDE4 (en + de), een top desktop manager
Xfce4 en + de), een middel gewicht desktop manager.

Fluxbox (en + de) is aanwezig op beide variaties als optionele lichtgewicht desktop manager.

Voor surfen op Internet, afhankelijk van de release variante, is er Konqueror en iceweasel.

Kantoorsoftware toepassingen, afhankelijk van de release variante : LibreOffice, Abiword, Gnumeric, tot bestandsbeheerders zoals Dolphin en Thunar, (om er maar enkele te noemen).

Om netwerk en internet connecties te configureren is er Ceni en voor WIFI zie onze WIFI Roaming documentatie.

Non-free driver informatie is hier.

Voor het partitioneren van de harddisk leveren we cfdisk en GParted gparted heeft een ntfs resize optie.

Diagnose tools zoals Memtest86+, een geavanceerde geheugen diagnose tool.

Alle ISO varianten bevatten een volledige set van command line interface tools. Een complete manifest van elke release variante is te vinden op elke download mirror: aptosid Mirrors, Downloaden en branden.
