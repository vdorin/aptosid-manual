ライブISOの内容

aptosid のライブISO には Debian の定義による dfsg-free のソフトウェアしか含まれていません。firmwareのnon-freeソースについての追加情報はこちらへ

このISOは(2012-12-01)Debian Sidに完全にもとづいており、aptosid独自のパッケージとスクリプトを加えて豊かに、より安定したものになっています。カーネルはLinux の3.6.9-rc1 バニラ・カーネルに基づいて最新かつ強力な独自のaptosidカーネルを使っています。 ACPIとDMAはデフォルトでオンになっています。

すべての種類の ISO には通常必要なコマンドラインツールがすべて揃っています。それぞれの ISO に搭載されているソフトウェアの完全なリストはダウンロードミラーサイトのマニフェストファイルを見てください: aptosid Mirrors, Downloading and Burning.

いろいろな ISO リリース

aptosid はDebian Sid システムへの導入点として5種類の最新ISOリリースを提供しています。インストールは通常10分から20分程度しかかかりません。

リリースの種類とは:
  1. KDE lite 32 bit , live-ISO 約600MB:
    KDE4 デスクトップの最小限のアプリケーションのみ搭載しています。その他のアプリケーションが長期的に必要であれば、aptシステムを使って簡単にインストールできます。
  2. KDE lite 64 bit , live-ISO 約600MB::
    KDE4 デスクトップの最小限のアプリケーションのみ搭載しています。その他のアプリケーションが長期的に必要であれば、aptシステムを使って簡単にインストールできます。
  3. XFCE 32 bit, live-ISO 約500MB:
    (最小限ではない）デスクトップシステム環境を搭載して、インストール後すぐに生産的な作業にとりかかれます。KDE4 より小さくシステムに負担をかけません。
  4. XFCE 64 bit, live-ISO 約500MB:
    (最小限ではない）デスクトップシステム環境を搭載して、インストール後すぐに生産的な作業にとりかかれます。KDE4 より小さくシステムに負担をかけません。
  5. KDE-full, 32 bit と+ 64 bitの複合ISO, live-ISO 約2.5GB:
    KDE4のアプリケーションスーツ全部を含み、16言語をサポートして、他にもLibreOfficeのオフィススーツやIceWeasel(FireFox)などのアプリケーションも搭載しています。

アプリケーションとツール

KDE4 とXFCE システムの最低必要条件

AMD64

  • 必要なCPU:
    • AMD64
    • Intel Core2
    • Intel Atom Series (2xx/ 3xx, N4xx/ D4xx/ D5xx; the N2xx series does not support EM64T)
    • VIA Nano
    • x86-64/ EM64TのできるCPU
    • それ以降の64ビット可能な AMD Sempron または Intel Pentium 4 CPUs (infobash -v3 のコマンドまたは /proc/cpuinfo で ”lm" フラッグが表示されるCPU).
  • 必要なRAMサイズ:
    • KDE: ≥512 MB RAM (なるべく≥1 GB RAM ), ライブAPTオプションによる言語サポートのためには≥1 GB RAM .
    • XFCE: ≥512 MB RAM.
  • 最低 640x480 ピクセルの解像度のあるVGAグラフィックカード
  • CD/DVDリーダーまたはUSBドライブ.
  • ≥3 GB HDD space, なるべく≥10+ GB
  • Supported boot methods:
    • PC-BIOS+MSDOS partition table (maximum HDD size <2 TB)
    • PC-BIOS+GPT partition table (maximum HDD size <8 ZB, /boot/ must physically be below 2 TB in its entirety)
    • UEFI+GPT (maximum HDD size <8 ZB)

i686

  • 必要なCPU:
    • Intel Pentium pro/ Pentium II
    • AMD K7 Athlon (K5/ K6 はダメ)
    • Intel Atom Series (Z5xx/ Z6xx, N2xx)
    • VIA C3-2 (NehemiahはOK, C3 Samuel または Ezra はダメ)/ C7
    • x86-64/ EM64TのできるCPU
    • i686 のフルコマンドセットが必要です
  • 必要なRAM:
    • KDE: ≥512 MB RAM (なるべく≥1 GB RAM), ライブAPTオプションによる言語サポートのためには≥1 GB MB RAM.
    • XFCE: ≥512 MB RAM.
    • With i686, only 3.1 to 3.4 GB RAM is accessible, amd64 (x86-64/ EM64T) is strongly recommended for 64 bit capable hardware.
  • 最低 640x480 ピクセルの解像度のあるVGAグラフィックカード
  • CD/DVDリーダーまたはUSBドライブ.
  • ≥3 GB HDD space, なるべく≥10+ GB
  • Supported boot methods:
    • PC-BIOS+MSDOS partition table (maximum HDD size <2 TB)
    • PC-BIOS+GPT partition table (maximum HDD size <8 ZB, /boot/ must physically be below 2 TB in its entirety)

アプリケーションとツール

aptosid-ISOには様々なプログラムとツールがあり、デスクトップ・ユーザが日々必要とするものは大部分カバーされています。

Choice of 2 major cutting edge Desktop Managers to access programs/applications:
KDE4 (en + de), 最先端のデスクトップマネージャー
Xfce4 (en + de), 中規模のデスクトップマネージャー

Fluxbox (en + de) は軽量デスクトップのオプションとしてどちらの種類のリリースにもふくまれています。

インターネットのサーフィングのためには、リリースの種類によってKonqueror または iceweaselが入っています。

Office Productivity applications (again depending on the ISO variation range from LibreOffice, Abiword, Gnumeric, through to file managers like Dolphin and Thunar, (to name only a few).

ネットワークやインターネット接続の設定のためにはCeni、無線LANの設定には WIFI Roaming を読んでください。

Non-free ドライバについては こちら.

ハードディスクのパーティションのためには cfdisk と GParted が搭載されています。GParted を使えば NTFS のサイズ変更もできます。

Memtest86+などの、メモリー診断ツールも含まれています。

すべての種類の ISO には通常必要なコマンドラインツールがすべて揃っています。それぞれの ISO に搭載されているソフトウェアの完全なリストはダウンロードミラーサイトのマニフェストファイルを見てください: aptosid Mirrors, Downloading and Burning.
