Privoxy

Privoxyとはプライバシー保護やウェッブページのデータ変更、クッキーの管理、アクセスコントロール、広告、バナー、ポップアップなど煩わしいのジャンクの除去といった進んだフィルタリングのできるウェッブプロクシーです。 Privoxyはとても柔軟に設定ができるため個々のユーザーの好みや必要に合わせてカスタマイズできます。 スタンドアローンモードとマルチユーザーモードの使い方が選べます。 PrivoxyはInternet Junkbuster(tm)をベースにしています。

Privoxyのインストール:

apt-get update
apt-get install privoxy

デフォルトのインストールはたいていのユーザーの出発点としてそのまますぐに役に立つと思います。 設定を調整する必要もあるでしょうが、必要の起こった時に徐々に変えていけばいいでしょう。 大抵の場合デフォルトの設定のままかちょっと手直しする程度で十分でしょう。

privoxyの設定ファイルから必要な行を選びコメント記号をとって有効にします。どの行が必要かはユーザーごとに違うでしょう。

Tor

Torは仮想トンネルのネットワークで、これを使ってユーザーやグループのネット上のプライバシーやセキュリティーを向上することができます。 ソフトウェア開発者はこれを使ってプライバシー保護がビルトインされたコミュニケーションツールを作ることもできます。 個人や組織がプライバシーを犠牲にせず公共のネットワークを使って情報を共有するアプリケーションをTorを基盤にして開発することができます。

代償としてインターネットの処理スピードが若干遅くなることも理解してください。

Torのインストール:

apt-get update
apt-get install tor

たいていの場合デフォルトの設定そのままで十分使えるでしょう。Tor をインターネット上で使うには Privoxy の設定が必要です。 Tor とインターネットのブラウジング を参照。

Tor の使用によってインターネットのスピードにマイナスの影響があることも知っておきましょう。

The Tor documentation には Tor に関するすべての事柄について説明があります。

Vidalia は Tor を使うソフトウェアの GUI コントローラで、Tor のスタート/ストップ、ステータスの一覧、バンドウィドスのモニターなどができます。

apt-get update
apt-get install vidalia
