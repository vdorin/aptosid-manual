!paste

Επικόλληση Debian (debian paste)

Το !paste θα γίνει ο φίλος σας στο κανάλι, διαφορετικά θα σας διώξουν για flooding. Έτσι εάν καλείστε να χρησιμοποιήσετε !paste, για να δώσετε το περιεχόμενο της κονσόλας σας, γράψτε !paste στο σημείο που συνομιλείτε (ή κάποιος μπορεί να έχει ήδη παράσχει την πληροφορία για να μπορείτε να ενεργοποιήσετε την επικόλληση (paste)), και αυτό θα επιστραφεί:

"paste" είναι 
http://paste.debian.net/:παρακαλώ μην
επιλέξτε κανέναν τονισμό σύνταξης, κάνετε την επικόλλησή (paste)
σας, κατόπιν δώστε τον σύνδεσμο (link) για την επικόλληση σας στο #aptosid.

Απλά επιλέγετε τη σύνδεση με ένα κλικ, που θα σας πάει σε έναν ιστό χώρο, επικολλήσετε το περιεχόμενο της κονσόλας που έχει ζητηθεί μέσα στο παράθυρο, κατόπιν πηγαίνοντας πίσω στην κορυφή, του παραθύρου επιλέξτε send.

Έπειτα θα δείτε ότι η είσοδός σας έχει προστεθεί στη βάση δεδομένων με διάφορες επιλογές για να συνδεθεί η επικόλληση με το IRC, που είναι η διεύθυνση που το κανάλι απαιτεί να δώσετε. (Παράδειγμα: Για να συνδεθείτε με την είσοδό (entry) σας: http://paste.debian.net/524). Η επικόλληση (paste) Debian μπορεί να ρυθμιστεί να δείχνει τις πληροφορίες για ένα συγκεκριμένο χρονικό διάστημα (μέχρι 72 ώρες). Έχετε επίσης την επιλογή να διαγράψετε τα paste / στοιχεία.

aptosid-paste

Το aptosid-paste θα επιτρέψει σε σας να επικολλήσετε την έξοδο (output) αρχείων από το τερματικό και tty, επομένως είναι ιδανικό εάν αντιμετωπίζεται προβλήματα στο init 3. Το aptosid-paste χρησιμοποιεί το http://rafb.net για σύνδεσμο και η έξοδος (output) είναι διαθέσιμη για 24 ώρες.

Μπορείτε να είστε ως απλός χρήστης (user) ή διαχειριστής (root) για να επιτραπεί το aptosid-paste, εντούτοις, σημειώστε ότι μερικά αιτήματα για την παραγωγή (output) απαιτούν την πρόσβαση διαχειριστή (root).

aptosid-paste <file>
ή
aptosid-paste -c "command1 --options parameters ; command2"
Παράδειγμα του aptosid-paste <αρχείο>
aptosid-paste /etc/fstab
 Για το paste δείτε εδώ: http://rafb.net/p/n8miQp85.html

Η διεύθυνση http://rafb.net/p/n8miQp85.html είναι αυτό που χρειάζετε να δώσετε στο #aptosid στο IRC

Παράδειγμα του aptosid-paste -c
aptosid-paste -c "fdisk -l; cat /etc/fstab"

Συνήθως είστε σε μη γραφικό περιβάλλον/init 3, επειδή πρόκειται να κάνετε μια dist-upgrade (βελτίωση) ή επειδή κάτι φοβερό έχει πάει στραβά με το σύστημά σας.

gpm

Ένα καλό εργαλείο να έχετε σε μη γραφικό περιβάλον είναι το gpm . Αυτό, θα επιτρέψει σε σας να χρησιμοποιήσετε το ποντίκι σας για να αντιγράφεται και να κολλάτε μεταξύ των τερματικών/κονσόλων.

gpm είναι ήδη ρυθμισμένο στο aptosid, αλλιώς κάντε: 

gpm -t imps2 -m /dev/input/mice

Εξασφαλίστε ότι το /etc/gpm.conf είναι υπαρκτό

service gpm restart

Τώρα είστε σε θέση να έχετε ένα ποντίκι σε μη γραφικό περιβάλλον (tty)

# CTRL-ALT-F2
$ aptosidbox login:  <όνομαχρήστη> <password> (όχι ως χρήστης root)
Κατόπιν γράψετε
$ aptosid-irc (αυτό ανοίγει το irssi)

# CTRL-ALT-F2
$ aptosidbox login:  <όνομαχρήστη> <password> (όχι ως χρήστης root)
Κατόπιν γράψετε
$ weechat-curses

/nick γράψτε_το_ψευδώνυμο_τις_επιλογής_σας

/join #aptosid

Εάν επιθυμείτε να συνδεθείτε σε έναν άλλο κεντρικό υπολογιστή (Server):

/server server.όνομα

Στην κατώτατη μπάρα, θα δείτε αριθμούς να εμφανίζονται εάν υπάρχει δραστηριότητα στα κανάλια, για να μεταπηδήσετε σε εκείνο το κανάλι, ALT-1 ALT-2 ALT-3 ALT-4 κτλ..

Για να βγείτε

/exit

# CTRL-ALT-F2

Infobash

Το infobash είναι ένα script πληροφοριών συστήματος, το οποίο είναι ανεξάρτητο από κάποιο συγκεκριμένο πελάτη IRC. Το script πληροφοριών συστήματος μπορεί να επιδείξει όλα τα είδη πληροφοριών που αφορούν το υλικό και το λογισμικό σας στους ανθρώπους στο κανάλι, έτσι μπορούν να σας βοηθήσουν να εντοπίσετε τα προβλήματα ή να θαυμάσουν τις προδιαγραφές του συστήματος σας και την έκδοση πυρήνα, όποτε θέλετε εσείς μέσα από την κονσόλα.

Για να χρησιμοποιήσετε το infobash script με το Konversation, γράψτε μέσα στο chat box:

/cmd infobash -v3

Για να χρησιμοποιήσετε το infobash script με άλλους πελάτες (clients) γράψτε μέσα στο the chat box:

/exec -ο infobash -v3
ή
/infobash -v3

Γράψετε σε μια κονσόλα:

infobash -v3

Διαβάστε το πλήρες σεμινάριο (tutorial) του infobash

man infobash
