Περιεχόμενα του Live ISO

Tο aptosid θα παρέχει μόνο dfsg-ελεύθερο λογισμικό στο live-cd, Παρακαλούμε ελέγχτε αυτό το link για επιπλέον πληροφορίες που αφορούν πηγές για μη ελεύθερο firmware.

Το ISO είναι παντελώς βασισμένο στο Debian Sid (μέχρι και την ημερομηνία 2010-06-13) εμπλουτισμένο και σταθεροποιημένο με πακέτα και scripts του aptosid. Έρχεται με τον τελευταίο εμπλουτισμένο πυρήνα του aptosid, ο οποίος είναι βασισμένος στον πιο πρόσφατο vanilla πυρήνα 2.6.3X.xx. Tα ACPI και DMA είναι ενεργοποιημένα εξ ορισμού.

Σε κάθε mirror υπάρχει μια ολοκληρομένη λίστα (manifest) των πακέτων που περιέχονται σε κάθε έκδοση: aptosid Mirrors, Κατέβασμα και κάψιμο.

Παραλλαγές εκδόσεων ISO

Το aptosid προσφέρει ως σημεία εισόδου στο Debian Sid 5 επίκαιρες live-ISO παραλλαγές έκδοσης, οι οποίες συνήθως δεν χρειάζονται περισσότερο από 10-20 λεπτά να εγκατασταθούν.

Οι παραλλαγές είναι:
  1. KDE lite 32 bit , live-ISO περίπου 600MB:
    περιέχει ένα μινιμαλιστικό σετ εφαρμογών για την επιφάνεια εργασίας KDE4. Η πρόσθεση περισσότερων εφαρμογών για να καλύψουν τις μακροχρόνιες ανάγκες σας γίνεται εύκολα μέσω apt.
  2. KDE lite 64 bit , live-ISO περίπου 600MB:
    περιέχει ένα μινιμαλιστικό σετ εφαρμογών για την επιφάνεια εργασίας KDE4. Η πρόσθεση περισσότερων εφαρμογών για να καλύψουν τις μακροχρόνιες ανάγκες σας γίνεται εύκολα μέσω apt.
  3. XFCE 32 bit, live-ISO περίπου 500MB:
    περιέχει ένα πλήρες Περιβάλλον Επιφάνειας Εργασίας (όχι μινιμαλιστικό), και έχει όλες τις εφαρμογές για να αρχίσετε δουλειά αμέσως. Καταναλώνει λιγότερους πόρους απ' ότι το KDE4.
  4. XFCE 64 bit , live-ISO περίπου 500MB:
    περιέχει ένα πλήρες Περιβάλλον Επιφάνειας Εργασίας (όχι μινιμαλιστικό), και έχει όλες τις εφαρμογές για να αρχίσετε δουλειά αμέσως. Καταναλώνει λιγότερους πόρους απ' ότι το KDE4.
  5. KDE-full, 32 bit + amd64 bit, live-ISO περίπου 2.5GB:
    περιέχει μια πλήρη σουίτα από KDE4 εφαρμογές με 16 υποστηριζόμενες γλώσσες και ανάμεσα σε άλλα, σουίτες εφαρμογών γραφείου όπως το LibreOffice και περιηγητές διαδικτύου όπως iceweasel (Firefox).

Εφαρμογές και εργαλεία

Ελάχιστες απαιτήσεις συστήματος KDE4 και XFCE

AMD64

  • Απαιτήσεις CPU:
    • AMD64
    • Intel Core2
    • Intel Atom 330
    • οποιοσδήποτε x86-64/ EM64T CPU ή νεότερος
    • νεότεροι 64 bit AMD Sempron και Intel Pentium 4 CPUs (ψάξτε για το "lm" flag μέσα στο /proc/cpuinfo ή χρησιμοποιείστε infobash -v3)
  • Απαιτήσεις RAM:
    • KDE: ≥512 MB RAM (≥1 GB RAM συνιστώνται), ≥1 GB RAM για liveapt.
    • XFCE: ≥512 MB RAM.
  • VGA κάρτα γραφικών με τουλάχιστον 640x480 pixel ανάλυση.
  • συσκευή οπτικού δίσκου ή USB συσκευή.
  • ≥3 GB χώρο στον HDD, συστήνονται ≥10 GB.

i686

  • Απαιτήσεις CPU:
    • Intel Pentium pro/ Pentium II
    • AMD K7 Athlon (όχι K5/ K6)
    • Intel Atom N-270/ 230
    • VIA C3-2 (Nehemiah, όχι C3 Samuel ή Ezra)/ C7
    • οποιοσδήποτε x86-64/ EM64T CPU ή νεότερος
    • χρειάζονται όλες οι i686 εντολές.
  • Απαιτήσεις RAM:
    • KDE: ≥512 MB RAM (≥1 GB RAM συνιστώνται), ≥512 MB RAM για liveapt.
    • XFCE: ≥512 MB RAM.
  • VGA κάρτα γραφικών με τουλάχιστον 640x480 pixel ανάλυση.
  • συσκευή οπτικού δίσκου ή USB συσκευή.
  • ≥3 GB χώρο στον HDD, συστήνονται ≥10 GB.

Εφαρμογές και Εργαλεία

Το aptosid-ISO περιλαμβάνει μια ποικιλία διαφορετικών προγραμμάτων και εργαλείων που καλύπτουν σχεδόν κάθε ανάγκη ενός desktop χρήστη στην καθημερινή του ζωή.

ΕChoice of 2 major cutting edge Desktop Managers to access programs/applications:
KDE KDE4 (en + de), ένας κορυφαίος διαχειριστής επιφάνειας εργασίας
Xfce4 (en + de), ένας σχετικά ελαφρής διαχειριστής επιφάνειας εργασίας.

Fluxbox (en + de) συμπεριλαμβάνεται και στις δυο παραλλαγές σαν ελαφρής διαχειριστής επιφάνειας εργασίας.

Για την περιήγηση στο διαδίκτυο, αναλόγος έκδοσης υπάρχει ο Konqueror και ο iceweasel.

Εφαρμογές γραφείου (και πάλι αναλόγος έκδοσης), ξεκινούν από το Koffice, Openoffice.org, Abiword, Gnumeric και φτάνουν μέχρι διαχειριστές αρχείων όπως τον Dolphin και τον Thunar.

Για την ρύθμιση δικτυακών και διαδικτυακών συνδέσεων υπάρχει το Ceni και για WIFI δείτε την τεκμηρίωση για το WIFI Roaming.

Πληροφορίες για μη-ελεύθερους οδηγούς μπορείτε να βρείτε εδώ.

Για την δημιουργία κατατμήσεων (partitions) στον σκληρό σας δίσκο, σας έχουμε εφοδιάσει με το cfdisk και το GParted. Το gparted έχει και δυνατότητα αλλαγής μεγέθους σε κατατμήσεις ntfs.

Παρέχονται διαγνωστικά Εργαλεία όπως το Memtest86+ - Advanced Memory Diagnostic Tool (Προχωρημένο Εργαλείο Διάγνωσης Μνήμης).

All ISO variations contain a full set of command line interface tools. Σε κάθε mirror υπάρχει μια ολοκληρομένη λίστα (manifest) των πακέτων που περιέχονται σε κάθε έκδοση: aptosid Mirrors, Κατέβασμα και κάψιμο.
