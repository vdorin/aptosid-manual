PHP Υποστήριξη για Apache2

Σιγουρευτείτε ότι έχετε όλες τις ακόλουθες γραμμές στο uncomment κάπου μέσα στο apache2.conf αρχείο σας (ή στο conf.d/php.ini σας):

LoadModule php5_module modules/libphp5.so

# Αναγκάστε το διερμηνέα PHP να χειρίζετε τα αρχεία με .php επέκταση.
AddType application/x-httpd-php .php

Εάν θέλετε να επιτρέψετε τους διαφορετικούς τύπους αρχείου δεικτών ελέγξτε για την ακόλουθη γραμμή στο αρχείο του /etc/apache2/apache2.conf 

DirectoryIndex index.html index.cgi index.pl index.php index.xhtml index.shtml

Ξαναξεκινήστε τον apache server

#/etc/init.d/apache2 restart
