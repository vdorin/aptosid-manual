Γράφοντας ένα aptosid ISO σε ένα USB stick, SSD κάρτα, shdc συσκευή με Linux, MS Windows™ ή Mac OS X™ Λειτουργικό Σύστημα

Ασχέτος με πιο λειτουργικό σύστημα χρησιμοποιήτε, οι ακόλουθες μεθόδοι θα σας επιτρέψουν να εγκαταστήσετε ένα aptosid ISO σε ένα USB stick, SSD κάρτα, shdc συσκευή (Secure Digital High Capacity).

Το aptosid ISO γράφεται πάνω στην συσκευή και σας επιτρέπει να έχετε το aptosid-on-a-stick αλλά δεν μπορείτε να έχετε το persist με αυτές τις μεθόδους.

Αν χρειάζεστε persist, το install-usb-gui είναι η συνηστόμενη μέθοδος εγκατάστασης όταν έχετε ήδη ένα aptosid σύστημα. Δείτε USB/SSD fromiso εγκατάσταση - aptosid-on-a-stick.

Προϋποθέσεις

  • Σιγουρευτείτε ότι το BIOS του υπολογιστή που θέλετε να εκκινήσετε με aptosid-on-a-stick/κάρτα μπορεί να εκκινήση από USB sticks ή SSD κάρτες. Τέτοιοι υπολογιστές είναι συνήθως αυτή που έχουν το πρωτόκολλο USB 2.0 και υποστηρίζουν boot from USB/SSD.
  • The USB/SSD should be recognised automatically and you should see the F4 menu option will say Hard Disk, otherwise invoke F4 >Hard Drive or add fromhd to the boot menu bootline.
  • Είναι σημαντικό να προσέξετε ότι οι παρακάτω μεθόδοι θα καταστρέψουν τον πίνακα κατατμήσεων της συσκευής προς εγραφή. Η απώλεια δεδομένων εξαρτάται από το μέγεθος του aptosid-*.iso. Δεν περιορίζει τον χώρο του δίσκου και ίσως καταφέρετε να ανακτίσετε τα δεδομένα τα οποία διέγραψε το ISO αλλά τα MS Windows φαίνεται να επιτρέπουν μόνο μια κατάτμηση. Παρακαλώ να είστε ασφαλής και να μην το κάνετε αυτό με τον 100+- gig δίσκο σας!

Λειτουργικά Συστήματα Linux

Συνδέστε το USB stick ή των αναγνώστη κάρτων με την κάρτα που θέλετε να γράψετε και:

cat /path/to/aptosid-*.iso > /dev/USB_raw_device_node

or

dd if=/path/to/aptosid-*.iso of=/dev/sdf
Παράδειγμα:

Συνδέστε την συσκευή και τρέξτε dmesg και κοιτάξτε το αποτέλεσμα:

sd 13:0:0:0: [sdf] Assuming drive cache: write through
sd 13:0:0:0: [sdf] Assuming drive cache: write through
sdf: sdf1 sdf2
sd 13:0:0:0: [sdf] Assuming drive cache: write through
sd 13:0:0:0: [sdf] Attached SCSI removable disk

Αν υποθέσουμε ότι κατεβάσατε ένα ISO με όνομα aptosid-2009-04-moros-kde-lite-i386-200912310312.iso (θα μπορούσατε να το μετονομάσετε σε aptosid-moros-kde-lite.iso) τότε η εντολή που πρέπει να τρέξετε είναι:

cat /home/username/aptosid-moros-kde-lite.iso > /dev/sdf

ή

dd if=/home/username/aptosid-moros-kde-lite.iso of=/dev/sdf

The USB/SSD should be recognised automatically and you should see the F4 menu option will say Hard Disk, otherwise invoke F4 >Hard Drive or add fromhd to the boot menu bootline.

MS Windows™ Συστήματα

Αυτό είναι απλό και γρήγορο. Εγκαταστήστε το flashnul, (Σελίδα για κατέβασμα).

Προσέξτε το όνομα του 'flashnul' όταν το αποσυμπιέσετε και εγκαταστήσετε γιατί μπορεί να φανεί σαν φάκελος με όνομα flashnul-x.x (για παράδειγμα, 'flashnul-1rc1.zip' κατεβασμένο, αποσυμπιεσμένο και εγκατεστημένο στο Program Files έχει ένα κατάλογο με όνομα flashnul-1rc1 ο οποίος έχει μέσα ένα κατάλογο με όνομα flashnul-1rc1, ο οποίος με την σειρά του περιέχει το πρόγραμμα flashnul και διάφορα αρχεία/καταλόγους που χρειάζεται.)

Συνδέστε το stick/κάρτα.

Εάν εγκαταστήσατε το flashnul στο Program Files και δεν είστε σίγουρος που είναι προσαρτημένο το USB stick, κοιτάξτε το αποτέλεσμα της εντολής 'flashnul -p'. Για να εκτελέσετε την εντολή κάντε κλικ στο run που βρίσκεται στο μενού και μετά γράψτε cmd για να ανοίξετε το τερματικό και μετά αλλάξτε καταλόγους και μπείτε στον κατάλογο του flashnul:

cd C:\Program Files\flashnul-1rc1\flashnul-1rc1\
flashnul -p

Το αποτέλεσμα θα είναι παρόμοιο με το πιο κάτω:

Available physical drives:
0       size = 60022480896 (55 Gb))
1       size = 2003828736 (1911 Mb)

Αν υποθέσουμε ότι κατεβάσατε ένα ISO με όνομα aptosid-2009-04-moros-kde-lite-i386-200912310312.iso (θα μπορούσατε να το μετονομάσετε σε aptosid-moros-kde-lite.iso), μεταφέρτε ή αντιγράψτε το στον flashnul-1rc1 κατάλογο ο οποίος περιέχει το πρόγραμμα και αρχεία/καταλόγους που χρειάζεται.

Όπως φαίνετε στο πιο πάνω αποτέλεσμα της εντολής 'flashnul -p' η USB συσκευή είναι προσαρτημένη στο 1. Γράψτε τα πιο κάτω και πατήστε enter (πιθανών να χρειάζεται να είστε διαχειριστής):

flashnul 1 -L aptosid-moros-kde-lite.iso

Σε MS Win7 πιθανών να χρειαστεί να δώσετε τα ονόματα των οδηγών, για παράδειγμα D: και C:

flashnul D: -L c:\flash\aptosid-2010-01-kde-lite.iso

The USB/SSD should be recognised automatically and you should see the F4 menu option will say Hard Disk, otherwise invoke F4 >Hard Drive or add fromhd to the boot menu bootline.

Mac OS X™ Συστήματα

Συνδέστε την USB συσκευή, το Mac OS X θα πρέπει να το προσαρτήσει αυτόματα. Μέσα στο Terminal (μπορείτε να το βρείτε στο Applications > Utilities folders), τρέξτε:

diskutil list

Βρείτε το όνομα της συσκευής και αποπροσαρτήστε τις κατατμήσεις της (ας υποθέσουμε ότι το όνομα είναι /dev/disk1):

diskutil unmountDisk /dev/disk1

Αν υποθέσουμε ότι κατεβάσατε ένα ISO με όνομα aptosid-2009-04-moros-kde-lite-i386-200912310312.iso (θα μπορούσατε να το μετονομάσετε σε aptosid-moros-kde-lite.iso) στο /Users/username/Downloads/, και η συσκευή USB έχει το όνομα disk1, τρέξτε:

dd if=/Users/username/Downloads/aptosid-moros-kde-lite.iso of=/dev/disk1

The USB/SSD should be recognised automatically and you should see the F4 menu option will say Hard Disk, otherwise invoke F4 >Hard Drive or add fromhd to the boot menu bootline.
