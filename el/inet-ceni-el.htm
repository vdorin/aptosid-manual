Σύνδεση με το διαδίκτυο / Πως να ρυθμίσετε το δίκτυo σας με το Ceni

Ceni is not a service. It is a configuration application that exists when configured, therefore does not use system resources as such.

Many wifi cards are deemed as non-free in Debian and cannot be shipped on an aptosid-iso*. Πιθανόν να χρειαστείτε μη-ελεύθερο firmware να είναι υπαρκτό πάνω σε ένα USB-stick για να το εγκαταστήσετε στο λειτουργικό σύστημα. Παρακαλώ δείτε μη-ελεύθερα firmware debs πάνω σε ένα stick.

Εάν έχετε διακομιστή (server) DHCP στο LAN (τοπικό δίκτυο) σας και ο υπολογιστής σας συνδέεται σε αυτόν, τότε οι ρυθμίσεις-δικτύου σας θα πρέπει να διαμορφώνονται αυτόματα κατά την εκκίνηση του υπολογιστή - διαφορετικά πρέπει να τρέξετε το Ceni επιλέξτε Kmenu > aptosid > Internet > Ceni. Αυτό θα ανοίξει ένα τερματικό/κονσόλα όπου θα σας ρωτήσει για τον κωδικό πρόσβασης υπερχρήστη root, (σε ένα live-cd δεν καθορίζεται κωδικός ).

Ο γρήγορος τρόπος να προσεγγιστεί το Ceni είναι να ανοίξετε το τερματικό/κονσόλα και να γράψετε

ceni

Όπου θα σας ζητηθεί ο κωδικός υπερχρήστη (root)

Σύνδεση με 56k Dial-up Modem

Το KDE έχει ένα "front end" για "dial up modem" το οποίο λέγεται KPPP Internet Dial-up Tool, και μπορείτε να το βρείτε στο Kmenu > Internet.

Η εφαρμογή περιέχει οδηγό βοηθείας ο οποίος παρέχει αναλυτικές οδηγίες για να ρυθμίσετε το modem σας και να καταφέρετε να συνδεθείτε με το διαδίκτυο.

Firewalls

Τα firewall συνήθως δεν χρειάζονται αν βρίσκεστε πίσω από ένα σωστά ρυθμισμένο δρομολογητή (router), παρ' όλα αυτά παίζουν πολύ σημαντικό ρόλο στη προστασία σας σε περίπτωση που θέλετε να συνδεθείτε στο διαδίκτυο με ένα "adsl usb modem" ή "dial-up modem".

ferm

ferm is a terminal tool to maintain complex firewalls, without having the trouble to rewrite the complex rules over and over again. ferm allows the entire firewall rule set to be stored in a separate file, and to be loaded with one command. The firewall configuration resembles structured programming-like language, which can contain levels and lists.

apt-get install ferm

ferm manual :: Please note that ferm will not teach you how a firewall works nor how to write good rules.

gufw

There is a GUI solution in the form of gufw which based on the backend application of ufw.

apt-get install gufw

The Debian description is :: gufw is an easy and intuitive way to manage your Linux firewall. It supports common tasks such as allowing or blocking pre-configured ports, common p2p or individual ports. It also supports IPv6 and many other options.
