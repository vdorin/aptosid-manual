aptosid-*.iso - Root-jelszó a aptosid LiveCD-n

Figyelmeztetés: Mielőtt a root felhasználó nevében teszel bármit, gondold végig, mit teszel! A web böngészéshez nincs szükség root felhasználóra.

aptosid-*.iso default password

A Sidux Live CD-n nincs root jelszó beállítva. Egy program root-jogokkal való futtatására többféle lehetőséged van:

Simple method, just type:

su
Egy (időszakos) root jelszó beállítása

Nyiss meg egy parancssort:

aptosid@0[aptosid]$ sudo passwd
Enter new UNIX password:
Retype new UNIX password:
passwd: password updated successfully
aptosid@0[aptosid]$

A beállított jelszót a gép újraindításáig lehet használni.

sudo is not supported on hard disk installations, see sudo and sux on how to set an alias.

Program futtatása root shell-ből

Gépeld a su-to-root -X -c <application> parancsot a terminálba, hogy root jogosultságot szerezz.

Parancssorban:

aptosid@0[aptosid]$ su-to-root -X -c gparted  (vagy bármilyen egyéb parancs...)

Néhány KDE alkalmazást a dbus-launch alkalmazással együtt kell elindítani:

dbus-launch <Alkalmazás>

Más lehetőség, amelyet a népszerűbb desktop környezetekben használhatsz:

Alt+F2 su-to-root -X -c <Alkalmazás>

Példa:

Alt+F2 su-to-root -X -c gparted

A root parancssor bezárásához gépeld be:

exit

vagy kattints a parancssor ablak jobb felső sarkába a bezáráshoz.

In case of a lockout on a aptosid-*.iso you need to do the following and follow the prompts to set a password:

alt+ctrl+F1
sudo passwd

Once the password is active, action the following:

alt+ctrl+F7

Programok telepítése a Live CD használata közben

apt-get update
apt-get install a-telepítendő-alkalmazás

Megjegyzés: A Live CD leállítása után a változások elvesznek, kivéve ha a fromiso és persist boot paraméterekkel használod.
